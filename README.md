# Python语言基础
## 课程介绍
Python是一门程序设计语言，作者是荷兰人Guido van Rossum，在1991年发布第一版。Python是动态类型语言，支持垃圾回收功能，支持面向对象、命令式、函数式和过程式编程。Python的设计哲学强调代码的简洁和可读性。相比Java，可以让开发者用更少的代码实现自己的想法。
